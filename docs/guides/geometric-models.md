# Geometric Models Guide

## Overview

Learn about geometric deep learning in Artifex. Geometric model implementations include point cloud, graph, mesh, voxel, and protein-specific modules in `models.geometric`.

- Point cloud processing
- Graph neural networks
- SE(3) equivariant networks
- Mesh generation

## Key Concepts

### Point Clouds

Unordered sets of 3D points with optional features.

### Equivariance

Networks that respect geometric symmetries.

### Graph Networks

Message passing on molecular and structural graphs.
