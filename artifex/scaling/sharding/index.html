    Sharding¤

    Module: generative_models.scaling.sharding

    Source: generative_models/scaling/sharding.py

    Overview¤

    Sharding strategies and parallelism configuration for scalable training.

    This module exposes the retained sharding strategies and configuration objects used by artifex.generative_models.scaling. It focuses on explicit strategy composition and mesh metadata instead of claiming a generic parameter-name to PartitionSpec inference layer.
